# PDF의 모든 것 — 기능 로드맵 100선

> 목적: "기능적으로 더 만들 것 100개"를 **우선순위와 함께** 정리.
> 원칙: **깔끔·실용·100% 브라우저 처리·무료** 정체성을 해치지 않는 것만 채택.
> 표기: **P0**=즉시 가치 큰 핵심 / **P1**=중요하지만 후순위 / **P2**=여유 있을 때 / **P3**=정체성과 충돌 가능, 보류 권장.

---

## A. 새 도구 (PDF 엔진 확장)
1. (P0) **PDF 압축** — 이미지 리샘플링/메타 제거로 용량 축소. *수요 1위, 미보유.*
2. (P0) **이미지 → PDF** (JPG/PNG 여러 장을 한 PDF로). *to-image의 역방향, 수요 높음.*
3. (P0) **PDF 회전** (페이지 단위 90/180/270, 저장).
4. (P1) **페이지 재정렬/이동** (드래그로 순서 변경 후 저장).
5. (P1) **워터마크 추가** (텍스트/이미지, 위치·투명도).
6. (P1) **페이지 자르기(Crop)** / 여백 제거.
7. (P1) **비밀번호 설정(암호화)** — unlock의 반대.
8. (P1) **PDF → 텍스트 추출**(복사용, OCR 아님).
9. (P2) **PDF 비교**(두 버전 차이 시각화).
10. (P2) **OCR**(스캔본 텍스트화) — 용량 큰 wasm 필요, 성능 검토.
11. (P2) **PDF → Word/HWP 변환** — 정확도 한계, 기대치 관리 필요.
12. (P2) **양식(Form) 채우기/평탄화**.
13. (P2) **북마크/목차 편집**.
14. (P2) **페이지 번호 매기기 고급**(섹션별, 로마자/한글).
15. (P3) **전자서명** — 법적 무게, 보류.

## B. 기존 도구 강화
16. (P0) **드래그&드롭 전 영역 활성화**(페이지 어디든 파일 드롭).
17. (P0) **여러 파일 일괄 처리**(합치기 외 도구에도 batch).
18. (P0) **결과 미리보기**(다운로드 전 썸네일 확인).
19. (P0) **처리 진행률·예상 시간** 표시 정교화.
20. (P1) **페이지 썸네일 그리드 선택**(추출/삭제 시 클릭 선택).
21. (P1) **실행취소/되돌리기**(편집 도구).
22. (P1) **분할 프리셋**(낱장/N장마다/북마크 기준).
23. (P1) **이미지 변환 옵션**(DPI, 포맷, 페이지 범위, 투명배경).
24. (P1) **파일명 규칙 지정**(다운로드 시 패턴).
25. (P1) **ZIP 묶음 다운로드**(여러 결과).
26. (P2) **합치기 시 파일 사이 빈 페이지/구분 삽입**.
27. (P2) **대용량 파일 스트리밍 처리**(메모리 관리).
28. (P2) **처리 후 자동 메모리 해제 표시**(신뢰 강화).

## C. UX / 인터랙션
29. (P0) **에러 메시지 한국어 친화화**(손상/암호 PDF 안내).
30. (P0) **키보드 단축키**(파일 열기, 실행, 도구 전환).
