# 도구별 개선 70선 (각 도구 10개)

> 7개 도구 × 10개 = 70. 클린·실용·100% 브라우저 처리 원칙 유지.
> 상태: [ ] 예정 · [x] 완료

## 1. merge — 합치기
1. [ ] 동일 파일(이름+크기) 중복 추가 방지
2. [ ] 손상/암호 PDF는 건너뛰고 안내(전체 실패 방지)
3. [ ] 파일별 페이지 수 표시(목록에 "n쪽")
4. [ ] 합계 요약(총 N개 · 합산 X페이지 · 용량)
5. [ ] "전체 비우기" 버튼
6. [ ] 이름순 자동 정렬 버튼
7. [ ] 출력 파일명 직접 지정
8. [ ] 파일 사이 빈 페이지 삽입 옵션
9. [ ] 진행률에 "n/총 파일" 표기
10. [ ] 결과에 출력 용량 표시

## 2. split — 분할
1. [ ] N페이지마다 분할 프리셋
2. [ ] 홀수/짝수 페이지 분리
3. [ ] 범위 입력값 총 페이지 초과 검증
4. [ ] 분할 결과 개수 미리 표시("→ 3개 파일")
5. [ ] ZIP 파일명 지정
6. [ ] 단일 범위면 ZIP 대신 단일 PDF로
7. [ ] "여기서 자르기" 그리드에 구간 색/번호 강화(기존 보강)
8. [ ] 모든 자르기 초기화 버튼
9. [ ] 범위 역순/정렬 정규화
10. [ ] 빈 범위·중복 범위 경고

## 3. unlock — 잠금해제
1. [ ] 업로드 시 암호 필요 여부 자동 감지 후 안내
2. [ ] 벡터 해제/이미지 해제 중 무엇이 쓰였는지 결과 표시
3. [ ] 이미지 해제 화질(배율) 선택
4. [ ] 비밀번호 표시/숨김 토글
5. [ ] 권한(인쇄/복사/편집) 제한 상태 표시
6. [ ] 이미지 해제 시 텍스트 손실 경고 강조(기존 보강)
7. [ ] 잘못된 비번 입력 시 친절 안내(기존 보강)
8. [ ] 처리 후 출력 용량/페이지 표시
9. [ ] 비번 입력 후 Enter로 실행
10. [ ] 결과 PDF 파일명에 원본명 반영

## 4. extract — 페이지 추출
1. [ ] 전체 선택 / 선택 해제 / 반전 버튼
2. [ ] 범위 입력 + 썸네일 선택 동기화(기존 보강)
3. [ ] 선택 개수 실시간 표시
4. [ ] 총 페이지 초과 입력 검증
5. [ ] 추출 순서 = 입력 순서 보장 안내
6. [ ] 출력 파일명 지정
7. [ ] 선택 0개일 때 실행 비활성 + 안내
8. [ ] 짝수/홀수 빠른 선택
9. [ ] 결과 페이지 수/용량 표시
10. [ ] 키보드로 페이지 토글(Enter/Space)

## 5. delete — 페이지 삭제
1. [ ] 전체 선택 / 해제 / 반전
2. [ ] 남는 페이지 수 실시간 표시("12 → 9쪽")
3. [ ] 모든 페이지 삭제 방지(기존 보강 + UI)
4. [ ] 총 페이지 초과 입력 검증
5. [ ] 짝수/홀수 빠른 선택
6. [ ] 출력 파일명 지정
7. [ ] 선택 0개일 때 실행 비활성 + 안내
8. [ ] 삭제 미리보기(빨간 표시) 강화
9. [ ] 결과 페이지 수/용량 표시
10. [ ] 키보드 토글

## 6. to-image — 이미지 변환
1. [ ] JPG 품질 슬라이더
2. [ ] PNG 투명 배경 옵션
3. [ ] 흑백(그레이스케일) 옵션
4. [ ] 배율(DPI) 선택(기존) + 예상 해상도 안내
5. [ ] 파일명 접두어 지정
6. [ ] 단일/ZIP 자동(기존 보강) + 개수 표시
7. [ ] 변환 범위 전체/특정(기존 보강)
8. [ ] 암호 PDF 안내(기존 보강)
9. [ ] 결과 총 용량 표시
10. [ ] 큰 배율 경고(메모리)

## 7. page-numbers — 페이지 번호
1. [ ] 글꼴 크기 선택
2. [ ] 글자 색 선택
3. [ ] 여백(가장자리 거리) 선택
4. [ ] 접두어/접미어 텍스트(예: "p.", "쪽")
5. [ ] 시작 페이지(어느 페이지부터 표기) 지정
6. [ ] 표지 제외(기존) + 미리보기 반영(기존 보강)
7. [ ] 위치 6종(기존) + 미리보기 위치 반영(기존 보강)
8. [ ] 형식 3종(기존) + "전체 N" 정확 계산(기존 보강)
9. [ ] 결과 용량 표시
10. [ ] 번호 배경(가독성용 반투명 박스) 옵션
