II

읽는 법 — 쓰는 법

How it works

「PDF의 모든 것」은 PDF 합치기, 분할, 잠금해제, 페이지 추출·삭제, JPG·이미지 변환, 페이지 번호 넣기까지 자주 쓰는 작업을 한곳에 모은 무료 도구 모음입니다. 설치나 회원가입 없이 브라우저에서 바로 시작하고, 파일은 서버로 올라가지 않으니 민감한 문서도 안심하고 다룰 수 있습니다.

① 도구를 고릅니다

목차에서 필요한 작업을 누르면 해당 도구 페이지로 이동합니다. 합칠지, 자를지, 번호를 넣을지 — 하고 싶은 일을 그대로 고르면 됩니다.

② 파일을 끌어다 놓습니다

화면에 PDF를 끌어다 놓거나 선택합니다. 이 순간에도 파일은 어디로도 전송되지 않습니다. 브라우저가 파일을 직접 읽어 작업을 준비합니다.

③ 결과를 내려받습니다

순서를 맞추거나 범위를 정한 뒤 실행 버튼을 누르면, 완성된 파일이 바로 내 기기에 저장됩니다. 워터마크도, 가입 유도도 없이 그대로 제출하면 됩니다.

스마트폰에서도

아이폰·안드로이드의 웹 브라우저에서도 별도 앱 설치 없이 동일하게 작동합니다. 메일로 받은 PDF나 스캔 앱으로 찍은 파일을 골라 그 자리에서 정리하세요.

자연스러운 한국어

어색한 기계번역 대신, 과제·보고서·계약서·민원처럼 실제로 자주 맞닥뜨리는 상황의 말투로 안내합니다. 처음 쓰는 분도 헤매지 않도록.

Q.정말 무료인가요?+
네, 전부 무료입니다. 결과물에 워터마크가 붙지 않고 파일 개수나 용량 제한도 없으며, 중간에 결제를 요구하지 않습니다.
Q.내 파일이 안전한가요?+
안전합니다. 모든 작업이 사용자의 브라우저 안에서 처리되며 파일이 서버로 전송되지 않습니다. 업로드 자체가 없으므로 외부에 파일이 보관될 일이 없습니다.
Q.프로그램을 설치해야 하나요?+
아니요. 설치나 회원가입 없이 웹페이지를 열기만 하면 바로 사용할 수 있습니다.
Q.어떤 작업을 할 수 있나요?+
PDF 합치기, 분할, 잠금해제, 특정 페이지 추출·삭제, JPG·이미지로 변환, 페이지 번호 넣기까지 일곱 가지 도구를 제공합니다.
Q.모바일에서도 되나요?+
네. 스마트폰이나 태블릿의 웹 브라우저에서도 별도 앱 설치 없이 동일하게 이용할 수 있습니다.
