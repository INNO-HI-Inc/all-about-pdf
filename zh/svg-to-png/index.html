    How to use

    SVG转PNG How to use

    1. 拖入或选择要转换的SVG文件，多个也能一次搞定。
    2. 设定输出倍率（1~4倍）。越大，PNG越清晰越大。
    3. 把背景选为透明或白色。
    4. 点转换按钮下载PNG。多张会打包成ZIP。
    您的文件绝不会上传到任何服务器。转换全程都在你的浏览器（设备）里完成，结束后一关闭页面即刻消失。任何文件都不会传到服务器，所以公司logo、未公开设计这类敏感SVG也能放心转换。
    说明

    这些场景很好用

    把设计工具（Figma、Illustrator）导出的logo SVG贴进演示文稿时，把图标传到微信·Notion时，或把网页用矢量导出成打印用高清PNG时，都很方便。只要调高倍率，同一个SVG就能从小缩略图到大横幅都不模糊。

    常见问题

    常见问题

    Q1为什么要把SVG转成PNG？
    SVG在微信、博客、一些文档编辑器等不支持矢量的地方可能显示不出来。转成PNG后，无论在哪儿都能像图片一样贴进去，透明背景也能原样保留。
    Q2画质会模糊吗？
    SVG是矢量，调高倍率也不会模糊。把输出倍率提到2~4倍，即便用于大屏或打印也能得到清晰的PNG。若只是小尺寸使用，1~2倍就够了。
    Q3会保存为透明背景吗？
    会。把背景设为“透明”，就会得到保留SVG透明部分的PNG。若需要白底，选“白色”即可。
    Q4能一次转换多个吗？
    可以，把多张SVG拖进来就能一次转换。单张直接下载，多张打包成ZIP。
    Q5有转换不了的SVG吗？
    引用了外部字体或网络图片的SVG，由于不向外部传输文件的策略，那部分可能显示为空白。字体方面，用事先转成轮廓（path）的SVG就能一模一样地呈现。
