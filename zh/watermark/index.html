    PDF 信息 提取文本

    提交用文档或分发资料，有时需要打上“内部机密”“副本”“草稿”这类标记。有些工具的中文水印会乱码，或要求把文件上传到服务器，敏感文档用起来让人不放心。这个工具输入文字后会把中文也清晰地渲染出来，叠加到所有页面，还能把logo·印章图片作为水印加入。你可以选择在中央打一个大大的斜向水印，或在整页像棋盘一样重复铺满以防非法复制。倾斜（0°·45°·90°）、大小、浓淡（浅~深）都可调，最重要的是文件不上传任何地方，只在你的浏览器里处理，合同、企划案这类文档也能放心打标记。

    PDF水印 How to use

    1. 把要加水印的PDF拖进来，或用[选择文件]上传。
    2. 选择“文字”或“图片”，指定要加的文字（如：内部机密）或logo图片。
    3. 选择排布（居中·棋盘）、倾斜、颜色·大小·浓淡。
    4. 点[添加水印]，下载所有页面都已标记的新PDF。
    您的文件绝不会上传到任何服务器。渲染文字、叠加水印并保存的整个过程都只在你的浏览器（设备）里完成，原始PDF和水印图片都不会传到任何服务器。所以内部机密文档、合同、内部企划案这类不便上传外部的文档，也能放心打上标记。
    说明

    这些场景很好用

    给对外分享的标书打上“内部机密”，为区分原件而标上“副本”“草稿”，或在分发资料上淡淡地叠一个公司logo时，都很好用。棋盘（重复）排布适合想遏制截图·再分发的内部文档，居中排布适合封面或合同这类需要一眼看清状态的文档。

    궁금한 점

    常见问题

    Q1中文水印也能不乱码地加上吗？
    能。像“内部机密”“禁止复制”这类中文文字都会被清晰地渲染加入。因为是把文字画成图像再叠加，即使收件人电脑上没有特定字体，显示效果也完全一样。
    Q2能把logo或印章图片作为水印加入吗？
    能。选择“图片”并上传PNG·JPG文件即可。用透明背景PNG（logo·印章）能不带背景自然地叠上去。浓淡和大小也可调。
    Q3会一次性加到所有页面吗？
    会，对文档所有页面同样应用。可以在中央大大地打一个，也可以在整页重复（棋盘）以防非法复制，两种任选。
    Q4水印会损伤原文文字或画质吗？
    不会。它是在原页面上叠加半透明标记，正文文字·画质保持原样。把浓淡设为“浅”，就能在不遮挡内容的前提下只留下标记。
    Q5真的免费吗？水印上会带广告吗？
    完全免费，除了你加的文字·图片，不会附带任何广告或服务标识。也没有使用次数·大小限制，不用注册。
