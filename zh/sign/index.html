    合同、同意书、各类表单需要签名或盖章时，就用这个工具。你可以用鼠标或手指直接手写签名，上传事先扫好的印章·签名图片，或输入姓名用字体生成。做好的签名可以拖到页面预览上想要的位置，抓住四角调整大小即可，也能在多个页面放多个。最重要的是，文件不上传任何地方，只在你的设备上处理，敏感合同也能放心签署。

    How to use

    PDF签名 How to use

    1. 把要签名的PDF拖进来，或用[选择文件]上传。
    2. 在手写·图片·输入中选一种，生成签名。
    3. 点[把签名放到页面上]，在预览里拖到想要的位置，用四角调整大小。
    4. 点[应用签名并下载]，下载带签名的新PDF。
    您的文件绝不会上传到任何服务器。生成签名并合并保存到PDF的整个过程都只在你的浏览器（设备）里完成，原始文件和签名图片都不会传到服务器。所以合同、同意书这类不便上传外部的文档，也能在无需担心上传的情况下签署。
    说明

    这些场景很好用

    适合给邮件收到的合同·同意书签好名直接回复时，想跳过打印→签字→再扫描的流程时，或需要把公司公章、名章干净地盖到文档上时。没有打印机和扫描仪，也能在屏幕上直接完成签名，而且文件不上传外部，敏感文档也能放心处理。

    常见问题

    常见问题

    Q1这是有法律效力的电子签名吗？
    这个工具是把签名图片绘制到PDF上的方式，与基于数字证书的电子签名（数字签名）不同。它很适合用于日常同意书·确认书·表单提交等，作用相当于手写签名。若是法律约束力很重要的合同，请遵循对方要求的电子签名方式。
    Q2印章图片怎么准备？
    把印章盖在白纸上拍照或扫描，再抠掉背景做成透明PNG上传，就能干净地放上去。若直接用白底图，会连方形背景一起放进去，所以尽量用透明背景的PNG。
    Q3一份文档能在多处签名吗？
    能。做好签名后，多次点[把签名放到页面上]就能放置多个，还能滚动页面，在想签的每一页都放上签名。每个签名的位置和大小都能单独调整。
    Q4签名位置或大小放错了怎么办？
    下载前，都可以在预览里重新拖动签名或抓住四角改大小，也能用签名上的 ✕ 按钮删除。原件不动，只把结果另存为新文件，尽管调到满意为止。
    Q5真的免费吗？有水印或次数限制吗？
    完全免费，不加水印。没有使用次数、文件大小限制，也不用注册。需要时尽管随时给PDF加签名。
