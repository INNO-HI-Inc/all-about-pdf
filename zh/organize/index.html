    PDF页面整理 How to use

    1. 拖入要整理的PDF,或点击[选择文件]上传。
    2. 所有页面以缩略图铺开后,拖动页面即可改变顺序。
    3. 歪向一侧的页面用 ↺ ↻ 旋转,想去掉的页面用 ✕ 删除。
    4. 点击[整理页面],下载整理好的PDF。
    您的文件绝不会上传到任何服务器。页面重排、旋转、删除的全过程都只在你的浏览器(设备)内完成,任何文件都不会上传到服务器。重新排列合同、诊断书、身份材料这类敏感文档的页面时,也能毫无上传顾虑地整理。
    说明

    这些场景很好用

    扫描仪把一两张扫歪、方向混乱时,多次扫描导致页面顺序错乱时,或想在提交前去掉中间的空白页、备注页时,都很适合一次整理好。不必在合并、拆分、删除等工具间来回切换,所见即所得,拖动着排好就完成了。

    궁금한 점

    常见问题

    Q1怎么改变页面顺序?
    把缩略图拖到想要的位置放下,顺序就会改变。用鼠标可抓住页面拖动,触屏设备上长按再拖即可。页面下方的数字是它在原文件中的页码,一眼就能看出正在移动哪一页。
    Q2可以只旋转某一页吗?
    可以。用每页的 ↺(向左)、↻(向右)按钮就能只把那一页每次转90度。点上方的“全部 ↺”“全部 ↻”则一次旋转所有页面。想只挑出扫歪的页面纠正方向时很方便。
    Q3重排、旋转、删除能一次一起做吗?
    可以。本工具把这三件事放在一个界面里一起处理。改好顺序、旋转部分页面、删掉不需要的页面后,只需下载一次,就能得到一份包含所有更改的PDF,不必反复切换多个工具。
    Q4不小心整理错了,能撤销吗?
    点上方的“恢复初始”按钮,顺序和旋转就会回到刚上传文件时的状态。而且原文件绝不会被改动,整理结果会另存为新的PDF,原文件始终安全。
    Q5页数很多的文档也能用吗?
    能。不过为避免页面太长,前120页会以缩略图显示,靠后的页面即使不在屏幕上显示,也会按原顺序原样包含在结果中。若要对超长文档的后半部分也做精细重排,建议先用[拆分]分开再整理。
    Q6旋转过的页面,文字或画质会受损吗?
    旋转不是把页面重新渲染成图片,而是只给PDF加上“方向”信息,因此文字仍可选中、复制,画质也不会下降。在保留原有清晰度的同时,只纠正方向。
