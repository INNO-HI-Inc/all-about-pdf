    How to use

    图片转PDF How to use

    1. 上传要做成PDF的图片(JPG、PNG),支持多张一次上传。
    2. 在列表中用 ↑↓ 或拖动来调整页面顺序。
    3. 在“按图片尺寸”和“适配A4”之间选择页面大小。
    4. 点击[生成PDF],下载合并好的一个PDF。
    您的文件绝不会上传到任何服务器。把图片合并成PDF的全过程都只在你的浏览器(设备)内完成,任何文件都不会上传到服务器。身份证复印件、合同材料这类敏感图片,也能毫无上传顾虑地做成PDF。
    说明

    这些场景很好用

    常见问题

    常见问题

    Q1可以把多张图片做成一个PDF吗?
    可以。上传多张图片后在列表里排好顺序,每张图片会依次占一页,合成一个PDF。页面顺序可以用 ↑↓ 按钮或拖动来调整。
    Q2支持哪些图片格式?
    支持JPG和PNG。手机照片(JPG)或屏幕截图(PNG)直接上传即可。其他格式(HEIC、WebP等)请先另存为JPG、PNG再上传。
    Q3“按图片尺寸”和“适配A4”有什么区别?
    “按图片尺寸”会根据每张图片的分辨率生成对应大小的页面,原样收入、不裁切。“适配A4”会把所有图片按比例居中放到A4纸上,适合需要统一尺寸用于打印、提交的场景。
    Q4照片画质会下降吗?
    会把原图原样收入PDF,不会随意压低画质。不过在“适配A4”下把小图片放大时,受原图分辨率所限可能会显得模糊。
    Q5有文件数量或容量限制吗?
    没有数量和容量限制,也没有水印。不过一次合并大量高分辨率照片会占用较多设备内存,如果变慢,可以分批制作。
