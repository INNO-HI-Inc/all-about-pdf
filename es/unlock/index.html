    Quita las restricciones de impresión y edición de tus propios PDF. Si el documento pide una contraseña al abrirlo, introduce la que ya conoces para desbloquearlo. Los archivos se procesan solo dentro del navegador y no se envían a ningún servidor.

    Tres pasos rápidos

    Desbloquear PDF How to use

    1. Sube el archivo PDF que quieras desbloquear.
    2. Si el documento solo tiene bloqueada la impresión o la edición, la restricción se quita al momento.
    3. Si pide contraseña al abrirlo, introduce la que ya conoces.
    4. Selecciona «Desbloquear como imagen» para eliminar la contraseña.
    5. Guarda en tu dispositivo el PDF ya desbloqueado.
    Tus archivos nunca se suben a ningún servidor.Esta herramienta no es un crack para averiguar contraseñas, sino una utilidad para eliminar una contraseña que ya conoces o quitar restricciones de impresión y edición. No se pueden abrir contraseñas que desconoces. El archivo y la contraseña que introduces se usan solo en tu navegador y no se guardan ni se envían a ningún servidor.
    Nota

    Cuándo te vendrá bien

    Úsala para quitar las restricciones de impresión o edición de un informe o trabajo que hayas creado tú y poder retocarlo de nuevo, para imprimir y archivar un folleto o una factura que hayas recibido por correo, o para abrir sin contraseña un contrato tuyo cuya clave conoces. Todo ello limitado a documentos sobre los que tengas permiso.

    Nota

    Ten en cuenta al desbloquear como imagen

    Cuando «desbloqueas como imagen» un documento con contraseña de apertura, cada página se convierte en una imagen y la contraseña desaparece. Con este método el documento se ve igual, pero después resulta difícil seleccionar y copiar el texto directamente o modificar el contenido. Es más seguro conservar aparte el PDF original con contraseña.

    Preguntas frecuentes

    Preguntas frecuentes

    Q1¿Puedo desbloquear un PDF si no conozco la contraseña?
    No. No se pueden quitar contraseñas que desconoces. Esta herramienta sirve para eliminar una contraseña que ya conoces o para quitar restricciones de impresión y edición; no es un crack de contraseñas.
    Q2¿Se guarda o se envía a algún sitio la contraseña que introduzco?
    No. La contraseña que introduces se usa solo dentro de tu navegador. No se guarda ni se envía a ningún servidor y, al terminar el proceso, desaparece de la memoria.
    Q3¿Qué diferencia hay entre una restricción de permisos y una contraseña de apertura?
    Con una restricción de permisos el documento se abre, pero está bloqueada la impresión, la edición o la copia; se puede quitar directamente sin contraseña. La contraseña de apertura pide una clave desde el momento de abrir el documento: introduce la que ya conoces y selecciona «Desbloquear como imagen» para eliminarla.
    Q4¿Quitar la contraseña de un PDF tiene algún problema legal?
    No hay problema si se trata de tus propios documentos o de documentos sobre los que tienes un derecho legítimo. Por ejemplo, puedes usarla para quitar la restricción de impresión de un folleto informativo que hayas recibido o de un contrato o informe que hayas creado tú. No debe usarse para desbloquear documentos ajenos sobre los que no tienes permiso.
    Q5¿Hace falta registrarse o instalar algún programa?
    No. Puedes usarla directamente, sin registro ni instalación. Es totalmente gratis, no añade marca de agua y no hay límite de número de archivos ni de tamaño.
    Q6¿Puedo desbloquear con seguridad escaneos o contratos sensibles?
    Sí. Los archivos se procesan solo en el navegador de tu dispositivo y no se suben a ningún servidor, así que documentos sensibles como contratos, trámites o escaneos no salen al exterior. A diferencia de las herramientas extranjeras que suben el archivo y lo borran pasado un tiempo, aquí directamente no se sube nada.
