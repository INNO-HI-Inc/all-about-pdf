    Convertir PDF a JPG How to use

    1. Sube el archivo PDF que quieras convertir.
    2. Elige el formato, JPG o PNG.
    3. Define la escala de calidad (de 1x a 3x).
    4. Selecciona todas las páginas o las que quieras.
    5. Pulsa el botón de conversión y descarga las imágenes. Si son varias, se reciben en un ZIP.
    Tus archivos nunca se suben a ningún servidor.Todas las conversiones ocurren dentro de tu navegador (tu dispositivo) y, al terminar, desaparecen en cuanto cierras la pantalla. Por eso puedes convertir en imagen con tranquilidad incluso documentos sensibles, como contratos o trámites.
    Nota

    Cuándo te vendrá bien

    Va bien para llevar una página de un trabajo o un informe al blog, para capturar como foto un gráfico de una presentación, o para enviar por WhatsApp una hoja concreta de una solicitud o un contrato. Puedes convertir directamente desde el móvil y compartir al momento, allí donde estés.

    궁금한 점

    Preguntas frecuentes

    Q1¿También convierte a PNG?
    Sí, basta con elegir PNG en el formato. Puedes seleccionar el formato que quieras entre JPG y PNG para la conversión. Se guarda en un solo formato cada vez.
    Q2¿Se conserva la calidad?
    Si subes la escala de calidad, sale más nítido. 1x es la calidad básica; subiéndola a 2x-3x, el texto y los gráficos se ven más definidos. Para capturar o imprimir una presentación, prueba a subir la escala.
    Q3¿Se guarda una imagen por cada página?
    Sí, cada página se convierte en una imagen. Si son varias, se reciben agrupadas en un archivo ZIP; si es una sola, se descarga directamente. También puedes elegir solo páginas concretas para convertir.
    Q4¿Cuál elijo, JPG o PNG?
    Si necesitas fondo transparente o el documento tiene mucho texto, el PNG queda más limpio. Para reducir el tamaño y subirlo ligero a WhatsApp o al blog, el JPG va mejor. Para compartir fotos, normalmente con JPG basta.
    Q5¿Hay límite de número de archivos o de tamaño?
    No hay límite de número ni de tamaño. Tampoco se añade marca de agua. Puedes convertir en imagen incluso informes con muchas páginas tal cual.
