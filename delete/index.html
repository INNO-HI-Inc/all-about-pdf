PDF 페이지 삭제

회원가입·설치 없이 무료로, 필요 없는 페이지만 쏙 빼서 새 파일로 저장하세요.

페이지를 삭제할 PDF를 끌어다 놓으세요

파일 선택

또는 끌어다 놓기 · 파일은 내 브라우저에서만 처리됩니다

    스캔하다 들어간 빈 페이지나 보고서 끝의 불필요한 장, 이제 번호만 입력해 지우세요. 삭제할 페이지를 빼고 깔끔한 PDF로 새로 저장합니다. 워터마크도, 개수 제한도 없습니다.

    PDF 페이지 삭제 사용 방법

    1. ① PDF 파일을 화면에 올려요.
    2. ② 삭제할 페이지 번호를 입력해요(예: 2, 4, 6-8).
    3. ③ 미리보기로 남길 페이지를 확인해요.
    4. ④ 삭제 버튼을 눌러요.
    5. ⑤ 페이지가 빠진 새 PDF를 내려받아요.

    파일은 서버로 전송되지 않습니다.올리신 PDF는 서버로 전송되지 않습니다. 모든 처리는 사용자의 브라우저, 즉 내 기기 안에서만 이뤄집니다. 그래서 계약서·청약 서류·민원 문서처럼 민감한 파일도 외부에 올리지 않고 페이지를 삭제할 수 있습니다.

    이런 분께 편리해요

    과제나 보고서를 스캔하다 빈 페이지가 끼었을 때, 계약서에서 서명만 받은 마지막 장을 빼야 할 때, 민원 서류 중 필요 없는 안내 페이지를 정리할 때 쓰기 좋습니다. 번호만 정확히 적으면 원하는 페이지를 빠르게 제거합니다.

    삭제 전 확인하면 좋은 점

    삭제는 미리보기에서 남길 페이지를 한 번 확인한 뒤 진행하세요. 번호를 잘못 적어도 원본은 그대로라 다시 시도할 수 있습니다. 양면 스캔본은 앞뒤 페이지 번호가 함께 들어가니 빼는 범위를 한 번 더 살펴보세요.

    자주 묻는 질문

    특정 페이지만 삭제할 수 있나요?
    네, 지우고 싶은 페이지 번호만 입력하면 됩니다. 예를 들어 3페이지만 빼고 싶으면 3을 입력하세요. 나머지 페이지는 순서 그대로 유지됩니다.
    여러 페이지를 한 번에 삭제하려면 어떻게 하나요?
    번호를 쉼표로 나눠 적거나 범위로 적으면 됩니다. 예를 들어 2, 4, 6-8처럼 입력하면 2·4페이지와 6에서 8페이지까지 한 번에 제거합니다.
    스캔할 때 생긴 빈 페이지만 골라 지울 수 있나요?
    빈 페이지를 자동으로 찾아주지는 않지만, 미리보기에서 비어 있는 페이지의 번호를 확인해 직접 입력하면 됩니다. 그 번호만 적어 삭제하면 빈 페이지를 깔끔하게 뺄 수 있습니다.
    원본 파일은 그대로 보존되나요?
    네, 원본은 손대지 않고 페이지가 빠진 결과를 새 파일로 저장합니다. 원본과 결과를 비교한 뒤 결정할 수 있어 안심하고 사용하세요.
    정말 무료인가요? 파일 개수나 용량 제한이 있나요?
    완전 무료이며 워터마크가 찍히지 않습니다. 파일 개수나 용량 제한, 회원가입도 없습니다. 필요할 때마다 자유롭게 페이지를 삭제하세요.
