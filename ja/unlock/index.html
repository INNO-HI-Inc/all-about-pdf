    How to use

    PDFロック解除 How to use

    1. ロックを外したいPDFファイルをアップします。
    2. 印刷・編集だけが制限された文書は、すぐに制限が外れます。
    3. 開くときにパスワードが必要な文書は、知っているパスワードを入力します。
    4. 「画像として解除」を選んでパスワードを取り除きます。
    5. ロックが外れたPDFをお使いの端末に保存します。
    ファイルはサーバーに送信されません。このツールはパスワードを割り出すクラックではなく、ご自身が知っているパスワードや印刷・編集の制限を取り除くためのツールです。知らないパスワードは外せません。ファイルと入力したパスワードはブラウザ内だけで使われ、サーバーへ保存・送信されることはありません。
    補足

    こんなときに便利です

    自分で作ったレポートや課題のPDFの印刷・編集制限を外して手直しするとき、メールで受け取った申込案内書や明細書を印刷して保管するとき、パスワードを知っている自分の契約書をパスワードなしで開いて見たいときにお使いください。いずれもご自身が権限を持つ文書に限ります。

    補足

    画像として解除するときに知っておいてください

    閲覧パスワードのかかった文書を「画像として解除」すると、各ページが画像に変わってパスワードが消えます。この方式は文書がそのまま見えますが、処理後は文字を直接選択・コピーしたり本文を修正したりするのが難しくなります。元のパスワード付きPDFは別に保管しておくほうが安全です。

    よくある質問

    よくある質問

    Q1パスワードを知らないPDFのロックを外せますか？
    いいえ。知らないパスワードは外せません。このツールはご自身が知っているパスワードを取り除いたり、印刷・編集の制限を外したりするためのもので、パスワードのクラックではありません。
    Q2入力したパスワードは保存されたり、どこかへ送信されたりしますか？
    いいえ。入力したパスワードはブラウザ内だけで使われます。サーバーへ保存されたり送信されたりすることはなく、作業が終わればメモリから消えます。
    Q3権限制限と閲覧パスワードは何が違いますか？
    権限制限は文書は開けるものの、印刷・編集・コピーが制限された状態で、パスワードなしですぐに外せます。閲覧パスワードは文書を開くときからパスワードを求めるロックで、知っているパスワードを入力し「画像として解除」を選んで取り除きます。
    Q4PDFのパスワード削除は法的に問題ありませんか？
    ご自身の文書、または正当な権限のある文書の制限を外す用途であれば問題ありません。たとえば、自分が受け取った申込案内書や、自分で作った契約書・レポートの印刷制限を外すときに使えます。権限のない他人の文書を外すために使ってはいけません。
    Q5会員登録やソフトのインストールは必要ですか？
    いいえ。会員登録もインストールも不要ですぐに使えます。完全無料で透かしも入らず、ファイル数や容量の制限もありません。
    Q6スキャン文書や機微な契約書も安全に外せますか？
    はい。ファイルはお使いの端末のブラウザ内だけで処理し、サーバーへアップしないので、契約書・行政手続きの書類・スキャン文書のような機微な文書も外部に漏れません。海外製ツールのようにアップロード後しばらくして削除する方式ではなく、そもそもアップロードしません。
