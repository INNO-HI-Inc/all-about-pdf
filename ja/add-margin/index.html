    製本すると綴じしろ側で文字が隠れてしまったり、印刷したら端が切れてしまった経験はありませんか。すべてのページの端に、狭い・普通・広いから選んだ白い余白を追加して、綴じしろを確保したり書き込みスペースを作ったりできます。中身のサイズはそのままに、まわりへ白い枠を足すだけの方式です。すべての処理はファイルをサーバーに送らず、お使いの端末内だけで完結するので、機密文書も安心して整えられます。

    How to use

    PDF余白追加 How to use

    1. 余白を追加したいPDFをドラッグ＆ドロップするか、[ファイルを選択]でアップロードします。
    2. 追加する余白の幅を、狭い・普通・広いから選びます。
    3. プレビューで余白が追加された仕上がりを確認します。
    4. [余白を追加]を押して、整ったPDFをダウンロードします。
    ファイルはサーバーに送信されません。余白を追加するすべての処理は、お使いのブラウザ（端末）の中だけで行われ、いかなるファイルもサーバーへ送信されません。契約書・提出課題・診断書のような人に見せたくない文書に余白を足すときも、外部にアップロードしないので、安心してPDF余白追加をご利用いただけます。
    補足

    こんなときに便利です

    印刷後に製本する際、綴じしろ側で文字が隠れないよう余白を確保したいとき、講義資料や会議文書に手書きメモのスペースを作りたいとき、プリンターが端を切ってしまうのを防ぎたいときに役立ちます。まわりに白い余白を足せば、製本・書き込み・印刷がどれもぐっとすっきりします。

    よくある質問

    よくある質問

    Q1余白をどのくらい追加するか選べますか？
    狭い・普通・広いの3段階から選べます。四辺に同じ幅で白い余白を追加するので、綴じしろや広い書き込みスペースが必要なら「広い」、ほんの少しだけ足したいときは「狭い」を選んでください。追加する前にプレビューで結果を確認できます。
    Q2片側の余白だけを追加できますか？
    このツールはページの四辺に同じ幅でまとめて余白を追加します。上・下・左・右をそれぞれ違う値で足す方式には対応していません。綴じしろを確保したり端の切れを防いだりするために、まわりを均等に広げたいときに最適です。
    Q3余白を追加すると文字や画像が小さくなりますか？
    いいえ。中身のサイズはそのままに、ページの外側へ白い枠を足すだけの方式なので、文字や画像は元のサイズのまま保たれます。ただしページ全体は余白のぶんだけ大きくなるので、紙に印刷する際は用紙サイズもあわせてご確認ください。
    Q4余白を追加すると画質が落ちますか？
    いいえ。ページを画像として描き直すのではなく、既存の中身のまわりに白い余白を足すだけの方式なので、文字の選択・コピーはそのまま使え、画質も落ちません。スキャン文書や写真入りの文書も鮮明さそのままです。
    Q5元のファイルはそのまま残りますか？
    はい。元ファイルには手を加えず、余白を追加した結果を新しいPDFとして別に保存します。仕上がりが気に入らなければ元ファイルからやり直せますし、透かしや件数・容量の制限もありません。
