PDF 합치기

  • 완전 무료 · 워터마크 없음

파일 선택

또는 끌어다 놓기 · 파일은 내 브라우저에서만 처리됩니다

    합치기 분할 잠금해제 페이지 추출 페이지 삭제 페이지 정리 이미지 변환 페이지 번호 이미지→PDF

    과제, 보고서, 계약서, 스캔본이 여러 장으로 나뉘어 있나요? 흩어진 PDF를 끌어다 놓고 순서만 정하면 한 파일로 합쳐집니다. 모든 작업이 내 기기 안에서 이뤄져요.

    사용 방법

    PDF 합치기 사용 방법

    1. PDF 파일들을 화면으로 끌어다 놓아요.
    2. 목록에서 ↑↓ 버튼으로 순서를 맞춰요.
    3. [합치기]를 눌러요.
    4. 하나로 묶인 PDF를 내려받아요.
    파일은 서버로 전송되지 않습니다.합치는 과정 전부가 내 브라우저(기기) 안에서만 이뤄지고, 창을 닫으면 흔적도 남지 않아요. 계약서나 민원서류처럼 민감한 문서도 마음 놓고 합칠 수 있습니다.
    참고

    이럴 때 쓰면 좋아요

    여러 장으로 스캔한 계약서를 한 파일로 묶을 때, 조별 과제를 각자 만든 부분끼리 합쳐 제출할 때, 청약이나 민원에 필요한 서류 여러 장을 하나로 정리할 때 편리합니다. 표지와 본문, 부록을 따로 만들었다면 순서만 맞춰 한 번에 합치세요.

    참고

    합치기 전에 확인하세요

    파일 이름이 비슷하면 순서가 헷갈릴 수 있으니 목록에서 ↑↓로 순서를 한 번 더 확인해 주세요. 비밀번호가 걸린 PDF는 먼저 잠금을 풀어야 합칠 수 있습니다.

    자주 묻는 질문

    자주 묻는 질문

    Q1한 번에 몇 개까지 PDF를 합칠 수 있나요?
    파일 개수나 용량에 제한을 두지 않았습니다. 보고서 챕터 여러 개나 스캔본 수십 장도 한 번에 합칠 수 있어요. 다만 파일이 아주 많거나 크면 기기 성능에 따라 시간이 조금 걸릴 수 있습니다.
    Q2JPG 같은 이미지도 PDF와 같이 합쳐지나요?
    이 도구는 PDF 파일끼리 합치는 기능입니다. 사진·이미지는 그대로 합쳐지지 않으니, 워드·한글·이미지 미리보기 등에서 '인쇄 → PDF로 저장'으로 먼저 PDF를 만든 뒤 이곳에서 합쳐 주세요. 그러면 한 파일로 깔끔하게 정리됩니다.
    Q3합치는 순서를 바꿀 수 있나요?
    네, 올린 파일 목록에서 ↑↓ 버튼으로 순서를 자유롭게 바꿀 수 있습니다. 표지, 본문, 첨부 순서처럼 원하는 대로 맞춘 뒤 [합치기]를 누르면 그 순서 그대로 한 파일이 됩니다.
    Q4아이폰이나 핸드폰에서도 PDF 합치기가 되나요?
    됩니다. 앱 설치 없이 휴대폰 브라우저에서 바로 사용할 수 있어요. 메일로 받은 PDF나 스캔 앱으로 찍은 파일을 골라 순서를 정한 뒤 합치면 됩니다.
    Q5합친 PDF에 워터마크가 찍히나요?
    워터마크는 전혀 찍히지 않습니다. 완전 무료이며 회원가입도 필요 없어요. 과제나 업무 문서에 그대로 제출해도 됩니다.
