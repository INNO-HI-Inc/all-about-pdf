PDF 페이지 추출

원하는 페이지만 골라 하나의 PDF로. 설치 없이 무료로 쓰세요.

✓ 서버 미전송 · 내 기기 처리 ✓ 완전 무료 · 워터마크 없음 ✓ 설치·회원가입 불필요

페이지를 추출할 PDF를 끌어다 놓으세요

또는 파일 선택 · 파일은 내 브라우저에서만 처리됩니다

    필요한 페이지만 따로 빼고 싶을 때 쓰는 도구예요. 추출할 페이지 번호를 입력하면 그 페이지만 모아 하나의 PDF로 저장합니다. 계약서 특정 조항이나 보고서 특정 장만 뽑아 제출할 때 편리해요.

    PDF 페이지 추출 사용 방법

    1. ① PDF 파일을 올려요.
    2. ② 추출할 페이지를 입력해요(예: 1, 3, 5-7).
    3. ③ 입력한 순서대로 페이지가 모이는지 미리 확인해요.
    4. ④ '추출하기'를 눌러 새 PDF로 저장해요.

    🔒파일은 서버로 전송되지 않습니다.올린 파일은 서버로 전송하지 않아요. 모든 추출 작업이 내 브라우저, 즉 내 기기 안에서만 이뤄지기 때문이에요. 계약서나 민원 서류처럼 민감한 문서도 외부로 나가지 않으니 안심하고 사용하세요.

    이럴 때 쓰면 편해요

    계약서에서 핵심 조항이 담긴 몇 쪽만 상대에게 보낼 때, 두꺼운 보고서에서 내가 맡은 장만 뽑아 제출할 때, 청약 안내문이나 민원 서류 중 필요한 페이지만 따로 보관할 때 유용합니다. 스캔본 PDF에서도 같은 방식으로 원하는 페이지만 골라낼 수 있어요.

    입력할 때 알아두면 좋은 점

    페이지 번호는 원본 PDF 기준으로 적어주세요. 존재하지 않는 번호(예: 20쪽 문서에 25)를 넣으면 그 부분은 추출되지 않습니다. 같은 페이지를 여러 번 적으면 그만큼 반복해서 들어가니, 결과 미리보기로 한 번 확인한 뒤 저장하면 실수를 줄일 수 있어요.

    자주 묻는 질문

    PDF에서 특정 페이지만 뽑을 수 있나요?
    네, 원하는 페이지 번호만 입력하면 그 페이지만 추출됩니다. 예를 들어 30쪽짜리 보고서에서 3쪽만 필요하면 '3'만 입력하면 돼요. 나머지 페이지는 결과 파일에 들어가지 않습니다.
    여러 구간을 한 번에 추출할 수 있나요?
    가능합니다. '1, 3, 5-7'처럼 쉼표로 페이지와 구간을 함께 적으면 한 번에 처리돼요. 떨어져 있는 페이지와 연속 구간을 섞어서 입력해도 됩니다.
    추출하면 페이지 순서가 유지되나요?
    입력한 순서를 그대로 따릅니다. '5, 1, 3'으로 적으면 5쪽, 1쪽, 3쪽 순서로 저장돼요. 원본 차례와 다르게 재배열하고 싶을 때도 활용할 수 있습니다.
    한 페이지씩 따로 저장하려면 어떻게 하나요?
    이 도구는 고른 페이지를 하나의 PDF로 모아줍니다. 페이지를 한 장씩 따로 떼어 여러 파일로 만들고 싶다면 'PDF 분할' 도구가 더 알맞아요. 일부만 모아 한 파일로 낼 때는 이 추출 기능을 쓰세요.
    파일이 서버에 올라가나요? 무료인가요?
    파일은 서버로 올라가지 않고 내 기기 안에서만 처리됩니다. 회원가입과 설치 없이 무료로 쓸 수 있고, 워터마크나 파일 개수·용량 제한도 없습니다.
