    양면 스캔을 하다 보면 글자 없는 뒷면이 통째로 빈 페이지로 끼어 들어가곤 합니다. 이 도구는 각 페이지를 하나씩 살펴 거의 흰 페이지만 골라내고, 글자나 그림이 조금이라도 있으면 그대로 남겨 둡니다. 비어 있는 페이지만 빠진 새 PDF로 저장되니 양면 스캔본의 빈 뒷면을 한 번에 정리하기 좋아요. 모든 처리는 파일을 서버에 올리지 않고 내 브라우저(기기) 안에서만 이뤄져, 민감한 문서도 안심하고 다듬을 수 있습니다.

    사용 방법

    PDF 빈 페이지 제거 사용 방법

    1. 빈 페이지를 정리할 PDF를 끌어다 놓거나 [파일 선택]으로 올려요.
    2. 각 페이지를 살펴 거의 흰 페이지를 자동으로 찾아내요.
    3. 빈 페이지로 표시된 결과를 미리보기로 한 번 확인해요.
    4. [빈 페이지 제거]를 눌러 새 PDF로 만들어요.
    5. 빈 페이지가 빠진 PDF를 내려받아요.
    파일은 서버로 전송되지 않습니다.빈 페이지를 찾고 제거하는 모든 과정은 내 브라우저, 즉 내 기기 안에서만 이뤄지고 어떤 파일도 서버로 전송되지 않습니다. 그래서 계약서·진단서·민원 서류처럼 민감한 스캔 문서의 빈 뒷면을 정리할 때도 업로드 걱정 없이 사용할 수 있습니다.
    참고

    이럴 때 쓰면 좋아요

    양면 스캔본에 빈 뒷면이 여러 장 끼었을 때, 문서를 한꺼번에 스캔하다 중간중간 빈 페이지가 섞였을 때, 제출이나 인쇄 전에 비어 있는 장을 한 번에 빼고 싶을 때 쓰기 좋습니다. 페이지 번호를 일일이 찾아 입력하지 않아도 자동으로 거의 흰 페이지를 골라내니 정리가 빠릅니다. 단, 아주 옅은 내용은 빈 페이지로 볼 수 있으니 결과를 한 번 확인한 뒤 내려받는 것을 권합니다.

    자주 묻는 질문

    자주 묻는 질문

    Q1빈 페이지를 어떻게 자동으로 찾나요?
    각 페이지를 하나씩 살펴 거의 흰 페이지를 빈 페이지로 골라냅니다. 글자나 그림이 조금이라도 있는 페이지는 내용이 있는 것으로 보고 그대로 남깁니다. 번호를 일일이 입력하지 않아도 비어 있는 페이지를 알아서 찾아 줍니다.
    Q2양면 스캔의 빈 뒷면 정리에 좋다는 게 무슨 뜻인가요?
    양면으로 스캔하면 인쇄가 없는 뒷면이 통째로 빈 페이지로 끼어 들어가는 경우가 많습니다. 이 도구는 그런 거의 흰 페이지를 찾아 빼 주기 때문에, 빈 뒷면이 여러 장 섞인 스캔본을 한 번에 깔끔하게 정리할 수 있습니다.
    Q3아주 옅은 글씨가 있는 페이지도 빈 페이지로 지워지나요?
    내용이 아주 옅으면 거의 흰 페이지로 보여 빈 페이지로 분류될 수 있습니다. 그래서 제거 전 미리보기에서 빈 페이지로 표시된 결과를 한 번 확인하시길 권합니다. 남기고 싶은 페이지가 섞였다면 확인 후 진행하면 됩니다.
    Q4원본 파일은 그대로 보존되나요?
    네, 원본은 손대지 않고 빈 페이지가 빠진 결과만 새 PDF로 저장합니다. 원본과 결과를 비교한 뒤 결정할 수 있어, 잘못 제거됐더라도 원본으로 다시 시도할 수 있습니다.
    Q5정말 무료인가요? 파일 개수나 용량 제한이 있나요?
    완전 무료이며 워터마크가 찍히지 않습니다. 회원가입이나 설치도 필요 없습니다. 모든 처리가 내 기기에서 이뤄지므로 필요할 때마다 자유롭게 빈 페이지를 정리하세요.
