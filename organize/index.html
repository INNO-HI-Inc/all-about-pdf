정리할 PDF를 끌어다 놓으세요

파일 선택

또는 끌어다 놓기 · 파일은 내 브라우저에서만 처리됩니다

    PDF를 올리면 모든 페이지가 아래에 펼쳐집니다. 끌어서 순서를 바꾸고, ↺ ↻로 회전, ✕로 삭제하세요.

    $> 파일은 이 페이지를 벗어나지 않습니다 · 100% 브라우저 처리

    스캔이 뒤집혀 들어간 페이지, 순서가 엉킨 보고서, 중간에 낀 빈 페이지를 따로따로 손볼 필요 없이 한 화면에서 정리하세요. 모든 페이지를 썸네일로 펼쳐 놓고, 끌어다 순서를 바꾸고, 옆으로 돌아간 페이지는 그 자리에서 회전하고, 필요 없는 페이지는 눌러서 빼면 됩니다. 번호를 외워 입력하지 않아도 눈으로 보며 정리하니 실수가 줄고, 모든 작업은 내 기기에서만 처리돼 민감한 문서도 안심하고 다듬을 수 있어요.

    사용 방법

    PDF 페이지 정리 사용 방법

    1. 정리할 PDF를 끌어다 놓거나 [파일 선택]으로 올려요.
    2. 모든 페이지가 썸네일로 펼쳐지면, 페이지를 끌어 순서를 바꿔요.
    3. 옆으로 돌아간 페이지는 ↺ ↻ 로 회전하고, 빼고 싶은 페이지는 ✕ 로 삭제해요.
    4. [페이지 정리하기]를 눌러 정리된 PDF를 내려받아요.
    파일은 서버로 전송되지 않습니다.페이지 순서 변경·회전·삭제 모든 과정은 내 브라우저(기기) 안에서만 이뤄지고, 어떤 파일도 서버로 전송되지 않습니다. 계약서·진단서·신분 서류처럼 민감한 문서의 페이지를 다시 배열할 때도 업로드 걱정 없이 정리할 수 있어요.
    참고

    이럴 때 쓰면 좋아요

    스캐너가 한두 장을 옆으로 찍어 방향이 뒤섞였을 때, 여러 번 스캔하느라 페이지 순서가 엉켰을 때, 제출 전 중간의 빈 페이지나 메모 페이지를 빼고 싶을 때 한 번에 정리하기 좋습니다. 합치기·분할·삭제 도구를 오가지 않고, 보이는 그대로 끌어서 맞추면 끝이에요.

    자주 묻는 질문

    자주 묻는 질문

    Q1페이지 순서를 어떻게 바꾸나요?
    썸네일을 끌어다(드래그) 원하는 자리에 놓으면 순서가 바뀝니다. 마우스로는 페이지를 잡아 옮기고, 터치 기기에서는 길게 눌러 끌면 됩니다. 페이지 아래 숫자는 원본에서의 쪽 번호라, 어떤 페이지를 옮기는지 한눈에 보여요.
    Q2한 페이지만 회전할 수 있나요?
    네. 각 페이지의 ↺(왼쪽) · ↻(오른쪽) 버튼으로 그 페이지만 90도씩 돌릴 수 있어요. 위쪽의 '전체 ↺' · '전체 ↻'를 누르면 모든 페이지를 한꺼번에 회전합니다. 스캔이 옆으로 들어간 페이지만 골라 바로잡을 때 편해요.
    Q3순서 변경·회전·삭제를 한 번에 같이 할 수 있나요?
    네. 이 도구는 셋을 한 화면에서 함께 처리합니다. 순서를 바꾸고, 일부 페이지를 회전하고, 필요 없는 페이지를 삭제한 다음 한 번만 내려받으면 모든 변경이 반영된 PDF가 만들어집니다. 도구를 여러 번 거칠 필요가 없어요.
    Q4실수로 잘못 정리했어요. 되돌릴 수 있나요?
    위쪽 '처음으로' 버튼을 누르면 순서와 회전이 파일을 처음 올렸을 때 상태로 돌아갑니다. 또한 원본 파일은 절대 바뀌지 않고, 정리 결과는 새 PDF로 따로 저장되니 원본은 그대로 안전합니다.
    Q5페이지가 아주 많은 문서도 되나요?
    됩니다. 다만 화면이 너무 길어지지 않도록 앞쪽 120페이지까지 썸네일로 보여 주고, 그보다 뒤의 페이지는 화면에 표시되지 않아도 원래 순서 그대로 결과에 포함됩니다. 아주 긴 문서의 뒷부분까지 세밀히 재배열하려면 먼저 [분할]로 나눈 뒤 정리하는 것을 권해요.
    Q6회전한 페이지의 글자나 화질이 깨지지 않나요?
    회전은 페이지를 이미지로 다시 그리는 것이 아니라 PDF에 '방향' 정보만 더하는 방식이라, 글자 선택·복사가 그대로 유지되고 화질도 떨어지지 않습니다. 원본 그대로의 선명함을 지키면서 방향만 바로잡아요.
