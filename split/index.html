    큰 PDF에서 필요한 부분만 나누고 싶을 때 쓰세요. 페이지를 1장씩 낱장으로 분리하거나, 1-3·4-8처럼 범위를 지정해 PDF 자르기가 됩니다. 결과는 ZIP으로 한 번에 받아요.

    사용 방법

    PDF 분할 사용 방법

    1. 분할할 PDF 파일을 올려요.
    2. '낱장으로 분리'나 '범위 지정' 중에서 골라요.
    3. 범위 지정이면 1-3, 4-8처럼 나눌 구간을 적어요.
    4. '분할하기'를 눌러요.
    5. 나뉜 파일을 ZIP으로 내려받아요.
    파일은 서버로 전송되지 않습니다.모든 처리가 100% 내 브라우저(기기) 안에서만 일어나요. 계약서·민원 서류·스캔본처럼 민감한 문서도 외부에 업로드하지 않으니 안심하고 PDF 나누기를 할 수 있습니다.
    참고

    이럴 때 쓰면 좋아요

    100쪽짜리 과제 모음에서 내 부분만 빼낼 때, 계약서를 조항별로 나눌 때, 스캔한 청약·민원 서류를 페이지별로 정리할 때 유용합니다. 두꺼운 보고서를 장별로 나눠 따로 보관하거나 메일에 첨부하기도 편해요.

    참고

    범위 지정 팁

    범위는 쉼표로 여러 개를 한 번에 적을 수 있습니다. 예를 들어 1-3, 4-8을 함께 넣으면 두 개의 파일로 나뉩니다. PDF를 반으로 자르고 싶다면 앞쪽 범위와 뒤쪽 범위를 각각 적으면 됩니다.

    자주 묻는 질문

    자주 묻는 질문

    Q1특정 페이지 한 장만 따로 저장하려면 어떻게 하나요?
    범위 지정에서 그 페이지 번호 하나만 적으면 됩니다. 예를 들어 '5'만 입력하면 5쪽만 따로 나옵니다. 여러 페이지를 골라 한 파일로 빼내는 추출 기능은 'PDF 페이지 추출' 도구를 이용하세요.
    Q2낱장 분리와 범위 지정은 뭐가 다른가요?
    낱장 분리는 모든 페이지를 1장짜리 파일로 전부 쪼갭니다. 범위 지정은 1-3, 4-8처럼 원하는 구간만 묶어서 나눕니다. 보고서를 장별로 쪼갤 때는 범위 지정이 편해요.
    Q3PDF를 분할하면 화질이 떨어지나요?
    아니요. 페이지를 그대로 떼어내는 방식이라 글자와 이미지가 원본 그대로 유지됩니다. 스캔본이나 사진이 들어간 문서도 화질 손상 없이 나뉩니다.
    Q4나눈 결과는 어떻게 받나요?
    나뉜 파일들을 ZIP으로 묶어 한 번에 내려받습니다. ZIP을 풀면 각 PDF 파일이 들어 있어요. 파일 개수나 용량 제한은 없습니다.
    Q5회원가입이나 설치가 필요한가요?
    필요 없습니다. 가입도, 프로그램 설치도 없이 브라우저에서 바로 PDF 자르기를 할 수 있어요. 워터마크도 찍히지 않습니다.
