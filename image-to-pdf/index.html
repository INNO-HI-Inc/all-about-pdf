    스캔한 사진, 휴대폰으로 찍은 서류, 화면 캡처 같은 이미지를 한 개의 PDF로 묶어 보세요. 여러 장을 끌어다 순서를 정하면 그대로 한 문서가 됩니다. 제출용 서류나 포트폴리오를 만들 때 편하고, 파일은 내 기기에서만 처리하니 안심하고 쓸 수 있어요.

    사용 방법

    이미지 PDF 변환 사용 방법

    1. PDF로 만들 이미지(JPG·PNG)들을 올려요. 여러 장을 한 번에 올릴 수 있어요.
    2. 목록에서 ↑↓ 또는 끌어다 페이지 순서를 맞춰요.
    3. 페이지 크기를 '이미지 크기대로' 또는 'A4에 맞춤' 중에 골라요.
    4. [PDF로 만들기]를 눌러 한 개의 PDF로 내려받아요.
    파일은 서버로 전송되지 않습니다.이미지를 PDF로 묶는 모든 과정은 내 브라우저(기기) 안에서만 이뤄지고, 어떤 파일도 서버로 전송되지 않습니다. 신분증 사본이나 계약 서류 같은 민감한 이미지도 업로드 걱정 없이 PDF로 만들 수 있어요.
    참고

    이럴 때 쓰면 좋아요

    휴대폰으로 찍은 서류 사진을 한 PDF로 묶어 제출할 때, 화면 캡처 여러 장을 보고용 문서로 만들 때, 손그림·포트폴리오 이미지를 한 파일로 정리할 때 편합니다. 휴대폰에서도 바로 만들어 그 자리에서 보낼 수 있어요.

    자주 묻는 질문

    자주 묻는 질문

    Q1여러 장을 한 PDF로 만들 수 있나요?
    네. 이미지를 여러 장 올린 뒤 목록에서 순서를 정하면, 각 이미지가 한 페이지씩 차례대로 들어간 하나의 PDF가 만들어집니다. 페이지 순서는 ↑↓ 버튼이나 끌어서 바꿀 수 있어요.
    Q2어떤 이미지 형식을 지원하나요?
    JPG와 PNG를 지원합니다. 휴대폰 사진(JPG)이나 화면 캡처(PNG)를 그대로 올리면 됩니다. 그 외 형식(HEIC·WebP 등)은 JPG·PNG로 먼저 저장한 뒤 올려 주세요.
    Q3'이미지 크기대로'와 'A4에 맞춤'은 무엇이 다른가요?
    '이미지 크기대로'는 각 이미지 해상도에 맞춰 페이지 크기를 만들어 잘림 없이 그대로 담습니다. 'A4에 맞춤'은 모든 이미지를 A4 용지 가운데에 비율을 유지해 배치해, 인쇄·제출용으로 크기를 통일할 때 좋아요.
    Q4사진 화질이 떨어지나요?
    원본 이미지를 그대로 PDF에 담기 때문에 화질이 임의로 낮아지지 않습니다. 다만 'A4에 맞춤'에서 작은 이미지를 크게 늘리면 원본 해상도 한계로 흐려 보일 수 있어요.
    Q5파일 개수나 용량 제한이 있나요?
    개수·용량 제한이 없고 워터마크도 없습니다. 다만 아주 많은 고해상도 사진을 한 번에 묶으면 기기 메모리를 많이 쓰니, 느리면 나눠서 만들어 보세요.
